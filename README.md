# Klasszicizmus kora
---
## Dallamvilág
- Világos, énekszerű dallamok
- Jól elkülönülő fő- és melléktémák
- Logikus felépítés
## Előadásmód
- Kontrasztok alkalmazása (hangos–halk, gyors–lassú)
- Kevesebb díszítés, mint a barokk zenében
- Világos ritmus és tempó
## A klasszikus 3
(mindhárom Bécsi származású)
1. Joseph Haydn (1732–1809
  - a szimfónia és a vonósnégyes „atyja”
  - kiegyensúlyozott, világos forma, sok humor
2. Wolfgang Amadeus Mozart (1756–1791)
  - zseniális dallamalkotás
  - operák, szimfóniák, versenyművek mestere
  - 5 éves kora óta irt zenét, 35 évesen meghalt :(
3. Ludwig van Beethoven (1770–1827)
  - átmenet a klasszicizmusból a romantikába
  - erőteljes érzelmek, drámai hangvétel
